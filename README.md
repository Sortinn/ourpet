## ourpet
A desktop pet app based on Android device
## 使用方法
1. fork项目使之成为一份属于自己仓库的副本;
2. 克隆项目（一定要从自己的账号下clone仓库，这样你才能推送修改）;
```
git clone https://github.com/$your-username/ourpet
```
3. 参与项目的开发;
4. 将项目更新推送到自己的仓库;
5. 发起pull request，将自己的更新与主仓库合并，使项目保持最新。
